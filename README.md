# Renamer

Partner named your kid Megatron? Didn't mean to call your farm Stardew Farm Farm? Called your kid Baby One and now they're a toddler? Worry not, Renamer adds a menu that allows you to change all of these names in a click.

Comes with a translation into every language the game comes with.

# Instructions

1. Download the mod
2. Unzip into your `Mods/` folder.
3. Open the menu. By default, this is done via the `;` (semicolon) key on your keyboard, the `Share` or `Back` keys on a controller, or through Iconic Framework or Better Game Menu (optional).
4. You can reconfigure the opening keys in the config file or through Generic Mod Config Menu.

# Compatible With

- **Better Game Menu:** open using the pencil menu tab.
- **Iconic Framework:** open using the pencil icon.
- **Android SMAPI** and **Cinderbox**
- **Multiplayer** (splitscreen and normal)
- **Translations:** the name of the menu and the mod's name in GMCM are automatically translated into the language you're using for every language included in the game.

# Permissions

Icon asset by [Pau](https://www.nexusmods.com/profile/suzukiPC) and may not be used.

This mod has a two line i18n. If you want to contribute a translation for a non-vanilla language, please send me the i18n file and I will add it into the mod with credit, but don't post an entire translation mod.

While this mod is open-source, I prefer that it not be reposted or forked (here or on any other platform) while I'm still active in the modding community.
